# 🎓 AI-Driven Feedback

**AI-Driven Feedback** — это современная образовательная платформа для изучения программирования на Python с использованием искусственного интеллекта. Платформа предоставляет мгновенную обратную связь на код, интерактивные уроки и справочные материалы для начинающих программистов.

## ✨ Основные возможности

- 📚 **Интерактивный учебный модуль** — пошаговое обучение Python от базовых концепций до продвинутых тем
- 🤖 **AI-тьютор** — получайте мгновенные советы и улучшения вашего кода
- 💬 **Встроенный чат-бот** — помощь и ответы на вопросы в режиме реального времени
- 📖 **Полный справочник Python** — быстрый доступ к синтаксису, функциям и методам
- 🎨 **Адаптивный интерфейс** — поддержка светлой и темной темы, работает на всех устройствах
- 💾 **LocalStorage** — ваши предпочтения (тема) сохраняются локально

## 🚀 Быстрый старт

### Способ 1: Локальный запуск
1. Клонируйте или загрузите репозиторий
2. Откройте файл `frontend/index.html` в любом современном браузере
3. Начните учиться!

### Способ 2: Использование локального сервера (рекомендуется)
```bash
# Для Windows (Python)
python -m http.server 8000

# Для macOS/Linux
python3 -m http.server 8000

# Затем откройте в браузере
http://localhost:8000/frontend/
```

## 🎨 Архитектура CSS

Использована переменная система CSS для упрощения управления темами:
- `--bg-primary` — основной фон
- `--bg-secondary` — вторичный фон
- `--text-primary` — основной текст
- `--text-secondary` — вторичный текст
- `--accent` — цвет акцента
- `--border-color` — цвет границ
