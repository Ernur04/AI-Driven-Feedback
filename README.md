# 🎓 AI-Driven Feedback

> **Инновационная обучающая платформа для эффективного изучения Python с поддержкой искусственного интеллекта**

**AI-Driven Feedback** — это современная, полностью интерактивная образовательная среда, разработанная специально для новичков, делающих свои первые шаги в программировании на Python. Наша платформа гармонично объединяет тщательно структурированные учебные модули, практические задания со встроенным онлайн-компилятором и умного AI-тьютора (на базе генеративных нейросетей), который моментально анализирует ваш код и предоставляет глубокие, персонализированные рекомендации по его улучшению.

Ключевой подход проекта — **«обучение через практику»**. Студент поэтапно изучает теоретическую базу, сразу же применяет полученные знания в редакторе кода и в ту же секунду получает фидбек. Прогресс наглядно отражается в удобном личном кабинете. В то же время, благодаря богатой системе мониторинга, преподаватели видят подробную статистику успеваемости каждого ученика, что позволяет гибко реагировать и адаптировать учебный процесс под нужды группы.

Одно из главных технических преимуществ нашей разработки — приложение работает **полностью на стороне клиента** (в браузере пользователя), эмулируя базу данных без необходимости подключения внешних серверов. Это делает систему невероятно быстрой, надёжной и доступной к запуску в любых условиях за пару кликов.

## 🚀 Быстрый старт

Развёртывание приложения максимально упрощено — вам не потребуется настраивать базы данных окружения!

1. **Клонируйте репозиторий на свой компьютер:**

   ```bash
   git clone https://github.com/Ernur04/AI-Driven-Feedback.git
   cd AI-Driven-Feedback
   ```

2. **Запустите проект:**
   Дважды кликните по файлу `frontend/index.html` и он откроется в вашем браузере по умолчанию.  
   _(В качестве альтернативы, вы можете запустить `run_server.bat` для старта полноценного локального сервера)_.

3. **Создайте учетную запись:**  
   Пройдите регистрацию как **Студент** (для погружения в обучение) или **Преподаватель** (чтобы оценить возможности панели мониторинга).

> **💡 Примечание:** Приложение работает автономно (Client-Side) — для его запуска и тестирования бэкенд не потребуется!
