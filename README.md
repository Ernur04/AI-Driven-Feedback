# 🎓 AI-Driven Feedback

**AI-Driven Feedback** — это современная образовательная платформа для изучения программирования на Python с использованием искусственного интеллекта. Платформа предоставляет мгновенную обратную связь на код, интерактивные уроки и персонализированную систему обучения для студентов и преподавателей.

## ✨ Основные возможности

- 📚 **Интерактивный учебный модуль** — пошаговое обучение Python от базовых концепций до продвинутых тем
- 🤖 **AI-тьютор** — получайте мгновенные советы и улучшения вашего кода
- � **Система Аутентификации** — полноценная регистрация и вход с разделением ролей (Студент/Преподаватель/Админ)
- 👤 **Личный кабинет** — управление профилем, загрузка аватаров и сохранение личных данных
- 📖 **Полный справочник Python** — быстрый доступ к синтаксису, функциям и методам
- 🎨 **Адаптивный интерфейс** — современный дизайн с поддержкой светлой и темной темы
- 💾 **LocalStorage DB** — эмуляция базы данных на стороне клиента для хранения пользователей и их прогресса

## 🚀 Быстрый старт

1. Клонируйте репозиторий.
2. Откройте `frontend/index.html` в браузере.
3. Зарегистрируйтесь как **Студент** или **Преподаватель**, чтобы получить доступ к полному функционалу.

## 🛠️ Технологии

- **Frontend**: HTML5, CSS3 (Glassmorphism), JavaScript (ES6+)
- **Icons**: Font Awesome 6.5.1
- **Fonts**: Google Fonts (Inter, Fira Code)
- **Database**: LocalStorage API (Client-side)
- **Стек**: Vanilla JS (без тяжелых фреймворков для максимальной скорости)

## 👥 Функционал ролей

| Роль              | Возможности                                                |
| ----------------- | ---------------------------------------------------------- |
| **Студент**       | Изучение теории, практика с AI, управление личным профилем |
| **Преподаватель** | Мониторинг прогресса, управление учебными материалами      |
| **Администратор** | Полное управление пользователями и системой                |
