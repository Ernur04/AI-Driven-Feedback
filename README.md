# 🎓 AI-Driven Feedback

> **Современная образовательная платформа для изучения Python с поддержкой искусственного интеллекта**

**AI-Driven Feedback** — это современная интерактивная образовательная платформа, разработанная специально для тех, кто только начинает свой путь в программировании на Python. Платформа объединяет в себе структурированные учебные материалы, практические задания с онлайн-компилятором и интеллектуального AI-тьютора, который мгновенно анализирует ваш код и даёт персонализированные советы по его улучшению.

В основе платформы лежит принцип **обучения через практику**: студент изучает теорию, сразу применяет знания в редакторе кода, получает фидбек от AI и видит свой прогресс в личном кабинете. Преподаватели и администраторы имеют доступ к детальной аналитике, что позволяет отслеживать успеваемость каждого студента и гибко управлять учебным процессом.

Платформа полностью работает на стороне клиента — **без необходимости в сервере или базе данных** — что делает её лёгкой в развёртывании и доступной для любого пользователя прямо из браузера.

## 🚀 Быстрый старт

1. **Клонируйте репозиторий:**

   ```bash
   git clone https://github.com/Ernur04/AI-Driven-Feedback.git
   cd AI-Driven-Feedback
   ```

2. **Откройте в браузере:**

   ```
   frontend/index.html
   ```

   Или запустите `run_server.bat` для локального сервера.

3. **Создайте аккаунт:**  
   Зарегистрируйтесь как **Студент** или **Преподаватель**, чтобы получить доступ к полному функционалу.

> **Примечание:** Проект работает полностью на стороне клиента — бэкенд не требуется.
