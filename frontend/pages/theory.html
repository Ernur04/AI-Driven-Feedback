Модуль 1 Python тілінің негізгі элементтері

Python — бұл жоғары деңгейлі, интерпретацияланатын бағдарламалау тілі. Оның синтаксисі адам тіліне (ағылшын тіліне) өте жақын, сондықтан оны үйрену оңай әрі қызықты. Бүгінде Python көмегімен жасанды интеллект (AI), веб-сайттар, Telegram боттар және күрделі деректер талдауы жасалады.

1.1 Экранға мәлімет шығару және түсініктемелер

Кез келген бағдарламалау тіліндегі алғашқы қадам — экранға мәлімет шығару. Бұл үшін Python-да print() функциясы қолданылады. Егер біз экранға мәтін шығарғымыз келсе, ол мәтінді міндетті түрде тырнақшаға (жалқы ' ' немесе қосарланған " ") алуымыз керек. Сондай-ақ, кодтың ішіне түсініктеме қалдыру үшін # (хэштег) белгісі пайдаланылады — оны компьютер оқымайды, ол тек программистің өзіне көмекші жазба.

print("Hello, World!") print("Python тілін үйренуді бастадым!") print(2026)

ЖИ Кеңесі: Егер сіз print("Сәлем) деп тырнақшаның бірін жабуды ұмытып кетсеңіз, бағдарлама жұмыс істемейді. Компьютер сізге SyntaxError: unterminated string literal деген қателік туралы хабарлама береді. Әрқашан жақшалар мен тырнақшалардың жұп болуын қадағалаңыз!

1.2 Айнымалылар және Мәліметтер типтері

Айнымалы (Variable) — бұл мәліметтерді сақтауға арналған компьютер жадындағы арнайы қорапша (контейнер) іспеттес. Біз оған ат беріп, ішіне кез келген ақпаратты сақтап қоя аламыз. Python-да айнымалыны жасау үшін теңдік = белгісі (меншіктеу операторы) қолданылады.

Python тілінде мәліметтердің бірнеше негізгі типі бар:

  • str (String) — Текстік мәтіндер (әрдайым тырнақшада жазылады).
  • int (Integer) — Бүтін сандар (оң, теріс немесе нөл).
  • float — Нақты (бөлшек) сандар (үтірдің орнына нүкте қойылады).
  • bool (Boolean) — Логикалық тип, тек екі мән қабылдайды: True (Ақиқат) немесе False (Жалған).
user_name = "Әлихан" user_age = 18 user_height = 1.75 is_student = True print(user_name) print(user_age)

Маңызды ереже: Айнымалының атын саннан бастауға болмайды (мысалы: 1name = "Асан" — қате). Сондай-ақ, аттардың арасында пробел болмауы тиіс. Егер аты бірнеше сөзден тұрса, төменгі сызықшаны қолданыңыз: user_score = 100.

1.3 Пайдаланушымен байланыс: input() функциясы

Бағдарлама түрі тек экранға ақпарат шығарып қана қоймай, пайдаланушыдан мәліметтерді қабылдай алуы керек. Ол үшін біз input() функциясын қолданамыз. Бұл функция экранға сұрақ шығарып, пайдаланушы пернетақтадан жауап беріп, Enter пернесін басқанша күтіп тұрады.

аты = input("Есіміңіз кім? ") print("Сәлем, " + аты + "!")

Есіңізде сақтаңыз: input() функциясы пайдаланушы сан енгізсе де, оны әрдайым мәтін (str) ретінде қабылдайды. Егер сізге ол санмен математикалық амал орындау керек болса, оны int() немесе float() функциялары арқылы санға айналдырып алуыңыз қажет. Мысалы: жас = int(input("Жасыңыз нешеде? "))

Модуль 2 Шарттар мен Тармақталу (Логикалық ойлау)

Өмірдегі сияқты, бағдарламалар да белгілі бір жағдайларға байланысты шешім қабылдауы керек. Мысалы: "Егер сыртта жаңбыр жауса — қолшатыр ал, әйтпесе — күннен қорғайтын көзілдірік ал". Осы логиканы іске асыру үшін Логикалық деректер типі (Boolean) қолданылады. Оның тек екі ғана мәні бар: True (Ақиқат) және False (Жалған).

2.1 Шартты операторлар: if, elif, else

Тармақталу процесі Python-да үш негізгі оператор арқылы жазылады:

  • if (егер) — ең алғашқы негізгі шартты тексереді;
  • elif (әйтпесе, егер) — бірінші шарт орындалмаса, қосымша басқа шарттарды кезекпен тексереді (бірнеше рет жазуға болады);
  • else (әйтпесе) — жоғарыдағы шарттардың ешқайсысы орындалмаған жағдайда ең соңында автоматты түрде іске асады.
жас = 16 if жас >= 18:     print("Сізге жүргізуші куәлігін алуға болады.") elif жас == 16:     print("Тағы 2 жыл күту керек, бірақ теорияны оқи берсеңіз болады!") else:     print("Сізге куәлік алуға әлі ерте.")

Маңызды ереже: Python-да шарттардан кейін міндетті түрде қос нүкте : қойылады, ал оның астындағы орындалатын код 4 бос орын шегініспен (Indent) жазылуы тиіс. Шегініс — бұл кодтың қай шартқа тиесілі екенін көрсететін Python-ның ең басты ерекшелігі. Егер шегініс дұрыс жасалмаса, бағдарлама IndentationError қатесін көрсетеді.

2.2 Салыстыру операторлары

if операторының ішінде шарт құрастыру үшін біз математикалық салыстыру белгілерін қолданамыз. Олардың нәтижесі әрдайым True немесе False болады:

  • > және < — үлкен және кіші
  • >= және <= — үлкен немесе тең, кіші немесе тең
  • == — тең бе? (Назар аударыңыз: жалғыз = белгісі мән беру үшін, ал қос == салыстыру үшін қолданылады)
  • != — тең емес пе? (мәндердің тең еместігін тексереді)
ұпай = 85 if ұпай == 100:     print("Керемет! Максималды нәтиже!") if ұпай != 0:     print("Сіз нөлден көп ұпай жинадыңыз.")

2.3 Күрделі шарттар: Логикалық операторлар (and, or, not)

Егер бізге бір уақытта бірнеше шартты тексеру керек болса, логикалық операторларды қолданамыз:

  • and (ЖӘНЕ) — барлық шарттар бір уақытта бірдей орындалғанда ғана іске асады.
  • or (НЕМЕСЕ) — берілген шарттардың кем дегенде біреуі орындалса жеткілікті.
  • not (ЕМЕС) — шарттың мәнін керісінше өзгертеді (True-ды False қылады, False-ты True қылады).
жас = 20 студент = True if жас >= 18 and студент == True:     print("Студенттік жеңілдік қолжетімді.") күн = "Жексенбі" if күн == "Сенбі" or күн == "Жексенбі":     print("Бүгін демалыс күні!")

2.4 Қысқаша шарт жазу (Тернарлық оператор)

Егер сізде өте қарапайым if...else құрылымы болса, кодты үнемдеу үшін оны бір жолға сыйдырып жазуға болады. Бұл бағдарламалауда Тернарлық оператор деп аталады.

сан = 7 нәтиже = "Жұп" if san % 2 == 0 else "Тақ" print(нәтиже)

Модуль 3 Қайталанулар (Циклдер)

Егер сізге экранға "Сәлем" сөзін 100 рет шығару керек болса, print() функциясын 100 рет көшіріп жазу тиімсіз. Мұндай бірсарынды, қайталанатын жұмыстарды автоматтандыру үшін циклдер (loops) қолданылады.

3.1 while циклі (Шартты цикл)

Python-дағы негізгі циклдердің бірі — while циклі. Ол «әзірше» немесе «шарт орындалып тұрғанша» деген мағынаны білдіреді. Түсінікті болу үшін ойын кейіпкерін елестетіңіз: кейіпкердің өмір саны (hp) 0-ден үлкен болып тұрған уақыт бойы ол картада жүгіріп, алға жылжи береді. Өмірі таусылған сәтте шарт False-қа айналады да, цикл тоқтап, "Ойын аяқталды" деген жазу шығады.

hp = 3 while hp > 0:     print("Кейіпкер алға жүгіруде! Қалған өмір:", hp)     hp = hp - 1 print("Шарт бұзылды (hp=0). Game Over!")

Сақ болыңыз: Егер сіз цикл ішінде шартты өзгертуді (мысалы, жоғарыдағы hp = hp - 1 кодын) ұмытып кетсеңіз, шарт әрдайым True болып қалады. Бұл "Шексіз циклге" (Infinite loop) әкеліп соғады және бағдарламаңыз қатып қалады. Мұндай жағдайда бағдарламаны тоқтату үшін терминалда Ctrl + C пернелерін басу керек.

3.2 for циклі және range() функциясы

Егер while циклі шартқа бағынса, for циклі қайталану саны алдын ала белгілі болғанда қолданылады. Ол белгілі бір сандар тізбегін немесе элементтер жиынтығын басынан аяғына дейін аралап шығады.

for циклімен бірге сандар тізбегін автоматты түрде жасау үшін range() функциясы жиі пайдаланылады:

  • range(5) — 0-ден 4-ке дейінгі сандар (0, 1, 2, 3, 4). Соңғы сан шекараға кірмейді!
  • range(1, 6) — 1-ден бастап 5-ке дейінгі сандар (басталуы және аяқталуы).
  • range(0, 10, 2) — 0-ден 9-ға дейінгі сандарды 2 қадаммен шығарады (0, 2, 4, 6, 8).
for i in range(5):     print("Сәлем!")

3.3 Циклді басқару: break және continue

Кейде циклдің жұмысын сырттай бақылап, оны мәжбүрлі түрде тоқтату немесе белгілі бір тұсын аттап өту қажет болады. Бұл үшін екі арнайы команда бар:

  • break — циклдің шарты әлі орындалып тұрса да, оны **толықтай тоқтатып**, циклден шығып кетеді.
  • continue — ағымдағы айналымды осы жерден үзеді де, астындағы кодтарды орындамай, бірден **циклдің келесі айналымына** көшеді.
for сан in range(1, 6):     if сан == 3:         break     print("Сан:", сан) for x in range(1, 6):     if x % 2 == 0:         continue     print("Тақ сан:", x)

Есіңізде сақтаңыз: break циклді біржолата жабады, ал continue тек сол сәттегі итерацияны өткізіп жіберіп, циклді әрі қарай жалғастырады.

Модуль 4 Деректер құрылымы (Мәліметтерді сақтау)

Осы уақытқа дейін біз бір айнымалыда тек бір ғана мәнді сақтап келдік (мысалы, жас = 16). Ал егер бізге сыныптағы 20 оқушының атын немесе дүкендегі тауарлар тізімін сақтау керек болса не істейміз? Әрқайсысына жеке айнымалы ашу ыңғайсыз. Осындай жағдайда бізге арнайы деректер құрылымдары көмекке келеді.

4.1 Тізімдер (Lists) — Өзгермелі контейнерлер

Тізім — бұл бірнеше элементті бір жерге жинақтап сақтауға арналған контейнер. Ол төртбұрышты жақшалармен [] белгіленіп, ішіндеғы мәліметтер үтір арқылы бөлінеді. Тізімнің ішіндегі мәліметтерді бағдарлама барысында өзгертуге, өшіруге немесе жаңадан қосуға болады.

  • .append(мәні) — тізімнің ең соңына жаңадан элемент қосады;
  • .remove(мәні) — тізім ішінен нақты көрсетілген элементті тауып, өшіреді;
  • Тізімдегі элементтерді рет-ретімен шақыру үшін индекстер қолданылады. Ең маңыздысы: бағдарламалауда санау әрқашан 0-ден басталады. Кері бағытта санау үшін теріс индекстер қолданылады (-1 — ең соңғы элемент).
инвентарь = ["Қылыш", "Қалқан", "Емдік сусын"] инвентарь.append("Алтын кілт") инвентарь.remove("Емдік сусын") print(инвентарь) print(инвентарь[0]) print(инвентарь[-1])

4.2 Кортеждер (Tuples) — Өзгермейтін тұрақтылар

Кортеж — бұл тізімге өте ұқсас, бірақ өзгертуге мүлдем болмайтын деректер жиынтығы. Ол дөңгелек жақшалармен () жазылады. Егер сіз кортеж жасасаңыз, оған жаңа мән қоса алмайсыз немесе ішіндегіні өшіре алмайсыз. Ол деректерді кездейсоқ өзгеруден қорғау үшін және бағдарлама жылдам жұмыс істеуі үшін керек.

апта_күндері = ("Дүйсенбі", "Сейсенбі", "Сәрсенбі") print(апта_күндері[0])

4.3 Жиындар (Sets) — Тек қана бірегей мәндер

Жиын — бұл элементтерінің орны реттелмеген (индексі жоқ) және ең бастысы — ішіндегі элементтері қайталанбайтын құрылым. Фигуралық жақшамен {} жазылады. Егер жиынға бірдей бірнеше элемент қоссаңыз да, ол тек біреуін ғана алып қалады. Тізімнен қайталанатын элементтерді тазарту үшін өте ыңғайлы.

сандар_тізімі = [1, 2, 2, 3, 3, 3, 4] бірегей_жиын = set(сандар_тізімі) print(бірегей_жиын)

4.4 Сөздіктер (Dictionaries) — «Кілт — Мән» жұбы

Сөздік — деректерді индекс бойынша емес, арнайы Кілт (Key) арқылы сақтайтын ең қуатты құрылымдардың бірі. Ол да фигуралық жақшамен {} жазылады, бірақ ішінде кілт: мән жұптары тұрады. Нақты өмірдегі телефон кітапшасы сияқты: адамның аты — кілт, телефоны — мән.

оқушы = {     "аты": "Аружан",     "жасы": 17,     "сыныбы": "11-А" } print(оқушы["аты"]) оқушы["ұпай"] = 95 print(оқушы)

Қорытынды естелік: Тізім [] — өзгертуге келеді (реттелген), Кортеж () — өзгертуге келмейді, Жиын {} — қайталанбайтын элементтер, Сөздік {кілт: мән} — мәліметтерді атаулы кілтпен сақтауға арналған.

Модуль 5 Функциялар және Жобалық жұмыс (Қорытынды деңгей)

Үлкен жобалар жазған кезде бір код фрагментін әртүрлі жерде бірнеше рет қайталап жазуға тура келеді. Кодты қайта-қайта көшірмеу үшін (DRY принципі — Don't Repeat Yourself) біз оны арнайы блокқа жинақтап, атын қоямыз. Бұл блокты Функция деп атайды.

5.1 Функция құрылымы және мән қайтару

Функцияны біз бір рет құрамыз да, кейін бағдарламаның кез келген жерінде тек атын атау арқылы шақыра береміз. Оның негізгі компоненттері:

  • def (define) — функция құруды бастайтын кілттік сөз;
  • Параметрлер мен Аргументтер — функция жұмыс істеуі үшін оған сырттан берілетін кіріс мәліметтері (жақшаның ішінде жазылады);
  • return — функцияның атқарған жұмысының соңғы нәтижесін сыртқа (негізгі бағдарламаға) қайтару командасы. return орындалған сәтте функция өз жұмысын бірден тоқтатады.
def есептеу_бонус(ұпай, коэффициент=1.0):     қорытынды = ұпай * коэффициент     return қорытынды ойыншы1 = есептеу_бонус(100, 1.5) ойыншы2 = есептеу_бонус(250) print("1-Ойыншы bonuses:", ойыншы1) print("2-Ойыншы bonuses:", ойыншы2)

Лямбда (Lambda) функциялары: Егер сізге тек бір жолдан тұратын шағын функция керек болса, анонимді лямбда функциясын қолдануға болады. Мысалы: шаршы = lambda x: x * x. Бұл кодты барынша қысқартуға көмектеседі.

5.2 Қорытынды Жоба: "Батырдың шытырман оқиғасы" RPG Симуляторы

Енді осы уақытқа дейін өткен барлық тақырыптарды (айнымалылар, шарттар, циклдер, тізімдер және функциялар) бір жүйеге біріктіріп, нақты шағын жоба жасап көрейік. Төмендегі кодты көшіріп, іске қосып көріңіз:

def жағдайды_тексеру(hp, инвентарь):     print("\n--- СІЗДІҢ СТАТУСЫҢЫЗ ---")     print("Өмір саны (HP):", hp)     print("Сөмкеңіздегі заттар:", инвентарь)     print("------------------------") def бастау_ойын():     hp = 100     инвентарь = ["Қылыш", "Нан"]     print("Ойынға қош келдіңіз, жас Батыр!")          while hp > 0:         жағдайды_тексеру(hp, инвентарь)                  print("Алдыңызда екі жол тұр: 1 - Үңгірге кіру, 2 - Жол жиегінде демалу, 3 - Ойыннан шығу")         таңдау = input("Қайсысын таңдайсыз? (1/2/3): ")                  if таңдау == "1":             print("Үңгірде айдаһар тап берді! Сіз жарақат алдыңыз (-40 HP).")             hp = hp - 40             if "Алтын" not in инвентарь:                 инвентарь.append("Алтын")                 print("Бірақ сіз үңгірден 'Алтын' тауып алдыңыз!")         elif таңдау == "2":             if "Нан" in инвентарь:                 print("Сіз нан жеп, күш жинадыңыз (+20 HP).")                 hp = hp + 20                 инвентарь.remove("Нан")             else:                 print("Сізде жейтін ештеңе қалмады.")         elif таңдау == "3":             print("Ойын тоқтатылды. Сау болыңыз!")             break         else:             print("Қате пәрмен! Қайта таңдаңыз.")                  if hp <= 0:             print("\nСіздің өміріңіз таусылды. Ойын аяқталды! Game Over.") бастау_ойын()
